# Triki - Tres en Raya

Juego经典的 Tic Tac Toe (Tres en Raya) implementado con React y Vite.

## Características

- 🎮 Modo 2 jugadores (local)
- 🤖 Modo contra computadora
- 📊 Historial de jugadas
- 🎨 Diseño moderno con animaciones
- ✨ Resalta la línea ganadora

## Estructura

```
src/
├── main.jsx
├── index.css
├── Game.jsx / Game.css
└── components/
    ├── Square.jsx / Square.css
    └── Board.jsx / Board.css
```

## Instalación

```bash
npm create vite@latest triki -- --template react
cd triki
npm install
npm run dev
```

## Tecnologías

- React 18
- Vite 5
- CSS puro (sin librerías)
